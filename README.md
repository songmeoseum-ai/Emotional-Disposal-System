# Emotional-Disposal-System

"Emotional Disposal System" 이라는 이름의 미니멀 웹앱을 만들어줘.

이 앱은 생산성 앱이나 일기 앱이 아니다.
사용자의 감정을 '기록'하는 것이 아니라 '폐기'하는 경험을 제공하는 프로젝트다.

감성적이거나 귀엽거나 따뜻한 느낌은 절대 금지.
차갑고, 무심하고, 기계적이며, 시스템 같은 분위기를 만들어라.

========================
파일 구조 (반드시 지킬 것)
========================
다음 3개의 파일로만 구성한다.

- index.html
- style.css
- script.js

프레임워크 사용 금지.
순수 HTML, CSS, JavaScript만 사용.

========================
핵심 컨셉
========================

사용자는 감정을 입력한다.
버튼을 눌러 감정을 처리한다.
그리고 모든 내용은 완전히 사라진다.

저장 없음.
기록 없음.
히스토리 없음.
복구 불가.

이 시스템은 말 그대로 아무것도 기억하지 않는다.

localStorage, DB, 쿠키 사용 금지.

========================
레이아웃 구성
========================

헤더 (중앙 정렬)
- 제목: Emotional Disposal System
- 부제: "We do not store your feelings."

메인 영역
- 큰 텍스트 입력창 (textarea)
- 가로로 배치된 버튼 3개
  - Trash
  - Shred
  - Launch

푸터
- 작은 문구:
  "Feelings are ephemeral."

모든 요소는 화면 중앙 정렬
최대 너비 520px
여백을 매우 크게 사용 (화이트스페이스 강조)

========================
디자인 스타일 가이드
========================

분위기:
- 차가움
- 무표정
- 정적
- 산업적
- 시스템 UI 느낌
- 감정 없는 기계 같은 느낌

금지:
- 밝은 색상
- 그라데이션
- 그림자
- 귀여운 요소
- 일러스트
- 이모지
- 과한 애니메이션

애플 설정 화면 + 터미널 + 전시회 같은 느낌으로 디자인

========================
컬러 팔레트
========================

배경: #FFFFFF 또는 #F5F5F5
기본 텍스트: #111111
보조 텍스트: #777777
테두리: #DDDDDD
포인트 색상: #2F2F2F 한 가지만 사용

거의 흑백에 가까운 모노톤 디자인

========================
폰트
========================

Pretendard / Inter / system-ui 같은
깔끔한 산세리프 폰트 사용

굵기:
Light 또는 Regular만 사용
Bold 금지

줄 간격 넓게
여백 넉넉하게
조용한 타이포그래피

========================
컴포넌트 스타일
========================

텍스트 입력창:
- 얇은 테두리
- 그림자 없음
- 포커스 시 살짝 색 변화
- 넓은 패딩

버튼:
- outline 스타일 (테두리만)
- 배경 투명
- hover 시 살짝 어두워짐
- 부드러운 전환 효과

카드 디자인, 과한 박스 UI 금지

========================
인터랙션 (중요)
========================

버튼 클릭 시:

1. 입력창 + 버튼 비활성화
2. 중앙에 "Processing..." 텍스트 표시
3. 2~3초 정도 느린 로딩 연출
4. 그 후:
   - 입력 내용 완전 삭제
   - 로딩 문구 사라짐
   - 다시 초기 상태로 복귀

주의:
- 성공 메시지 표시 금지
- "삭제 완료" 같은 문구 금지
- 아무 일도 없었던 것처럼 조용히 초기화

허무하고 맥 빠지는 느낌이 중요하다.

선택 효과:
- 페이드 아웃
- 아주 미세한 글리치
- 노이즈
- opacity 애니메이션

과한 효과 금지

========================
애니메이션
========================

CSS transition 사용
opacity, background, color 정도만
200~300ms
부드럽고 거의 느껴지지 않게

========================
자바스크립트 기능

- textarea 초기화
- UI 리셋

코드는 최대한 단순하고 깔끔하게 작성

- script.js 전체 코드
